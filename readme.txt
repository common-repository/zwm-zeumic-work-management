ZWM Zeumic Work Management, a free todo list WordPress plugin works Standalone or can work in tandem with WooCommerce to help you run your business.

== Description ==

ZWM Zeumic Work Management is a free todo list plugin that can work in tandem with WooCommerce to help you run your business. ZWM Zeumic Work Management allows you to manage and organise your staff, products or tasks, departments and clients.

It provides a fully featured todo list, which can either automatically be populated with WooCommerce order items, or can have todo items added manually.

= Manage staff =

With ZWM Zeumic Work Management, you can delegate tasks or products to staff, varying the priority of the products or tasks alongside adding meta, customer notes and internal notes.

= Privileges of staff =

ZWM Zeumic Work Management comes with the ability to adjust staff members' privileges, including the options for editing departments, users, priorities of products or task, clients, meta, customer notes and internal notes.

= Manage departments =

Different departments have different tasks or products and with ZWM Zeumic Work Management, you can create departments and organise your staff into departments to see certain products or tasks.

= Edit style =

You have control over the style of ZWM Zeumic Work Management. You can edit the width of department, users, priority, client, order number, meta, customer, notes, internal notes and actions.

== Installation ==

= Minimum Requirements =

* WordPress 4.4 or greater
* WooCommerce 3.0 or greater
* PHP 5.3 or greater
* MySQL 5.0 or greater
* Some payment gateways require fsockopen support (for IPN access)

= Automatic installation =

For an automatic download of ZWM Zeumic Work Management, log in to your WordPress and select the Plugins Menu. In the search bar, insert "ZWM Zeumic Work Management", select the plugin and click "Install Now".

For ZWM Zeumic Work Management setup, the staff are created via users, the departments and priorities created through ZWM Zeumic Work Management.

To add the ZWM Zeumic Work Management to a page, edit the page and add the shortcode [zwm_list] where desired.

[Optional] For the WooCommerce pathway of ZWM Zeumic Work Management, in the search bar, type "WooCommerce", click the plugin and press "Install Now". Navigate to the back end of WordPress and under ZWM Zeumic Work Management, click "Settings". Change the integration option to "WooCommerce" and click "Save Changes".

= Updating =

Ensure your site is backed up before automatically updating ZWM Zeumic Work Management.

If, by a small chance, there is an issue with the pages after an update, go to WordPress > Settings > Permalinks and click "save".
